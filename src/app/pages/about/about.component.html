Introducción


En el presente trabajo, realizaremos un desarrollo web con énfasis en el diseño Front-end que permitirá traer datos de una Interfaz de Programación de Aplicaciones (API application programming interface) por medio de consumo de servicios y que tiene como fin implementar todos los conceptos en la aplicación Angular.




Avances


En estas semanas de trabajo en Angular, realizamos la creación del repositorio y aprendimos el manejo de Git. La creación del proyecto la compartimos y la trabajamos con Visual Studio Code, en donde introducimos TypeScript, HTML, CSS. Realizamos la creación de los componentes y maquetamos la vista y elementos, para saber la estructura que llevaría el diseño de la pagina. Aprender a prueba y error el manejo de Angular fue complejo para todos, hay muchas etiquetas y comandos; al desarrollar es común que lo que no se visualice lo que digitamos, el desconocimiento de los recursos que contiene la plataforma nos llevo a consultar muchos mas fuentes de información. .




Entrega Final


Se cumple con el objetivo principal, se genero un catalogo web de los pokemón. Este cumple con los requisitos planteados inicialmente, se emplearon herramientas Angular, TypeScript, HTML y CSS. Se hace el consumo del API REST de acuerdo a la informacion solicitada; el consumo de API se evidencia en la muestra de cada Pokemón en la web, su informacion a traves de una tarjeta (Card), se creo una caja de busqueda que nos arroja la informacion escrita, en caso de esta ser mal digitada nos arroja un aviso de "No se encontro el Pokemon (Valor Digitado)". Logramos crear varias paginas dentro del App y enlazar una con otra de tal manera la pagina Web fuera amigable con el cliente o usuario final.

Los tiempos de ejecucion del cronograma no fue cumplido a cabalidad debido a la no participacion del agunos compañeros, lo que llevo a hacer algunos ajustes de ultimo momento. Sin embargo los estudiantes que integramos el grupo y trabajamos en el desarrollo del proyecto, dividimos las tareas para dar cumplimiento a las actividades dentro de los tiempos de cada Entrega (1, 2 y 3). Logramos coordinar horarios de trabajo para hacer pruebas, ajustes y los videos requeridos.
