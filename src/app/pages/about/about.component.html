Introducción


En el presente trabajo, realizaremos un desarrollo web con énfasis en el diseño Front-end que permitirá traer datos de una Interfaz de Programación de Aplicaciones (API application programming interface) por medio de consumo de servicios y que tiene como fin implementar todos los conceptos en la aplicación Angular.




Avances


En estas semanas de trabajo en Angular, realizamos la creación del repositorio y aprendimos el manejo de Git. La creación del proyecto la compartimos y la trabajamos con Visual Studio Code, en donde introducimos TypeScript, HTML, CSS. Realizamos la creación de los componentes y maquetamos la vista y elementos, para saber la estructura que llevaría el diseño de la pagina. Aprender a prueba y error el manejo de Angular fue complejo para todos, hay muchas etiquetas y comandos; al desarrollar es común que lo que no se visualice lo que digitamos, el desconocimiento de los recursos que contiene la plataforma nos llevo a consultar muchos mas fuentes de información. .




Entrega Final


Se cumple con el objetivo principal, de generar un catalogo web de los pokemón. Este cumple con los requisitos planteados inicialmente. Se emplearon herramientas Angular, TypeScript, HTML y CSS; logrando así el consumo del API REST. De acuerdo a la información solicitada; el consumo de API se evidencia en la muestra de cada Pokemón en la web, su información a través de una tarjeta (Card), se creo una caja de búsqueda que nos arroja la información escrita, en caso de esta ser mal digitada nos arroja un aviso de "No se encontró el Pokémon (Valor Digitado)".

Logramos poner muchas mas animaciones a los iconos, celdas y banner; asi como crear varias paginas adicionales como “about” “home” “ x filtro” “todos los pokemon en iconos” dentro del App y enlazar una con otra de tal manera la pagina Web fuera mas amigable con el usuario.

Los tiempos de ejecución del cronograma al final no fue cumplido debido a la no participación de algunos compañeros, lo cual llevo mas esfuerzo y dedicación tanto para aprender, como para realizar la entrega, esto se ve reflejado en los commits y su historial por cada integrante de la creación de cada modulo.

En estas semanas de trabajo en Angular, se aprendieron muchos comandos básicos que pueden ser aplicados en Java, PHP incluso en Python. La creación del proyecto nos dejo como conocimiento el manejo de GitHub Desktop y Visual Studio Code.

Aprender a prueba y error el manejo de Angular fue complejo, al desarrollar es común que no se visualice lo que digitamos, el desconocimiento de los recursos que contiene la plataforma nos llevo a consultar muchos mas fuentes de información.
